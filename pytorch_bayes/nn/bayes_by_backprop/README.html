Bayesian Learning

  • Prior

    p(w)

  • Likelihood

    p(y1:n|x1:n,w)=∏i=1np(yi|xi,w)

  • Posterior

    p(w|x1:n,y1:n)=1Zp(w)∏i=1np(yi|xi,w)

    where Z=∫p(w)∏i=1np(yi|xi,w)dw

  • Prediction

    p(y∗|x∗,x1:n,y1:n)=∫p(y∗|x∗,w)p(w|x1:n,y1:n)dw

Maximum Likelihood Estimation (MLE)

wMLE=arg⁡maxwlog⁡P(y1:n|x1:n,w)=arg⁡maxw∑i=1nlog⁡P(yi|xi,w)

Maximum a Posteriori (MAP)

wMAP=arg⁡maxwlog⁡P(w|x1:n,y1:n)=log⁡P(w)+arg⁡maxw∑i=1nlog⁡P(yi|xi,w)

Being Bayesian by Backpropagation (Bayes by Backprop)

F(y1:n|x1:n,θ)≈∑j=1NMC[log⁡q(w(j)|θ)−log⁡P(w(j))−∑i=1nlog⁡P(yi|xi,w(j))]

  • Variational Posterior Parameters

    θ=(μ,ρ)

  • Gaussian Variational Posterior

    w=μ+σ⊙ϵ

    where ⊙ is element-wise multiplication operator, σ=log⁡(1+eρ) and ϵ∼N(0,I)

  • Cost Function

    f(w,θ)=log⁡q(w|θ)−log⁡P(w)−∑i=1nlog⁡P(yi|xi,w)

    where using either Gaussian prior

    log⁡P(w)=∑klog⁡N(wk|0,σ02)

    or Gaussian scale mixture prior

    log⁡P(w)=∑klog⁡[πN(wk|0,σ12)+(1−π)N(wk|0,σ22)]

  • Update the Variational Posterior Parameters

    μ←μ−αΔμ

    and

    ρ←ρ−αΔρ
